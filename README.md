# Trabalhos Acadêmicos C

O presente repositório e uma coletânea de trabalhos acadêmicos, os artefatos desenvolvidos são:

- Campo Minado
- Jogo da Velha
- Mundo dos Blocos
- Ordenação
- Conjunto
- Tabela Hash
- Simulação Fábrica
